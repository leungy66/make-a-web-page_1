Intro of Programming Nanodegree Lesson Highlights

Lesson 1: The basic of the Web and HTML

1. About HTML

  • The full name of HTML is Hyper Text Markup Language.
  • Hyper Text = "The content of what people read"
  • Markup = "the format"/ "What is the page look like"
  • HTML can also make a page reference to other doc and link to other pages.

2. HTML Jargon

  • Tag: An HTML tag is always contained within angled brackets. Most tags have an opening tag and a closing tag. Some tags (called "void" tags) do not require a closing tag.
  • Element: An HTML element refers to everything within a set of opening and closing tags.
  • Attribute: This is a property of an HTML element.

Lesson 2: Creating a structured Document

1. Front end webpage coding structure with House analogy

  • HTML~set the structure of the house, where wall is, hwo many bedrooms
  • CSS~the style, what is the color of the wall
  • Javascript~ Tools for interaction of page element, TV remote

2. Relationship among HTML, CSS and DOM

  • HTML as Structure; CSS as style; DOM as Document Object Model
  • Code is organized in tree like structure for better follow-up and modifications.

3. Boxify Design

  • Convert page design to code with a boxify view
  • Major Boxes with Smaller Boxes inside

4. <div> and <span>

  • These are the two 'container' elements that can be used to build the structure of a web page.

5. About learning how to code

  • 4 Stages: Ignorance, awareness, ability, fluency
  • Feeling confuse and frustrate is normal, but don’t get disappointed and angry.
  • The gap between Novice and Expert:Scattered bunches of facts vs Structured understanding;and,ideas vs ability to execute

Lesson 3:Adding CCS style and HTML structure

1. Three ways to write CSS stuffs

  • In the <head> of HTML with style tag
  • In a separate CSS file
  • Inline with HTML

2.CSS selector

  • Allows programmer to target specific HTML elements and apply style to them.
  • ID selector: #id_value{}
  • Class selector: .class_value{}
  • Tag selector: tag_name{}
  • Attribute Selector: [attibute="value"]{}
  • Positional selector :nth-child or :nth-of-type

3. Basic steps for constructing webpage using HTML and CSS

  1. Figure out natural boxes on the design
  2. Identify repeated styles and semantic element
  3. Write HTML
  4. Apply styles (From bigger box to smaller box)
  5. Fix things

4. Avoiding Repetition is important

  • Avoid repetition when programming, it could be more convenience for coder to adjust and modify codes.
